Assessment Assistant

Project Description

The Assessment Assistant is a program designed to aid teachers by helping them manage a database of questions and creating a test and exporting it to a PDF. The program is divided into two sub-programs which are designed to perform different tasks. These two programs are the database manager and test manager. The project offers fully functional GUI and coded entirely on Java using Eclipse and the IText API.

Functionality

Database Manager

  • Adds, removes or edit multiple choice and short answer questions of different categories
  • Knowledge, thinking, application, communication
  • A binary search for folders and files
  • Alphabetical sort within folders
  • Saving folder system and opening a pre-existing one
